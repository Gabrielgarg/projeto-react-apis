## Como rodar esse projeto?

```bash
# Clone este repositorio
$ git clone linkrepo

# Acesse a pasta do projeto no seu terminal
$ cd projeto-pokedex

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run start
```
